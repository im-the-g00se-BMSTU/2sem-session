# 19. Многофайловая организация проекта. Файлы с расширением \*.h и \*.c, их назначение, пример многофайлового проекта.

Многофайловая организация проекта - это разделение программы на несколько файлов.

Обычно используют:

.c файлы - файлы с реализацией функций;

.h файлы - заголовочные файлы с объявлениями функций, типов, констант;

main.c - файл с функцией main.

Для чего проекты делятся на файлы:

- Простота понимания кода.
- Простота компиляции.
- Многоразовое использование функций
- Оптимизация памяти

Пример многофайлового проекта:

```c
void sortArray(int *arr, int size) {
    for (int i = 0; i < size - 1; i++)
        for (int j = 0; j < size - 1 - i; j++)
            if (arr[j] > arr[j + 1])
                swap(arr + j, arr + j + 1);
}
```

## файл main.c

```c
int main() {
    int arr[] = {5, 2, 4, 1, 3};
    int size = 5;

    sortArray(arr, size);
    printArray(arr, size);
    return 0;
}
```
