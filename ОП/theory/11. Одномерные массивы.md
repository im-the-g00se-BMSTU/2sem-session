Одномерный массив - это набор элементов одного типа, расположенных в памяти подряд. В СИ массивов НЕТ, но принято называть массивом непрерывный блок памяти, хранящий элементы одного типа.

1. Статическое выделение на стеке: int arr\[10\];
2. Динамическое выделение (в куче): int\* arr = (int\*)malloc(10 \* sizeof(int));
3. Динамическое выделение памяти на стеке: int arr\[SIZE\]; SIZE - какая-то константа в коде, либо ранее полученное, известное на момент выделения памяти, число.

Передача самого массива в функцию невозможна, можно передать указатель на первый элемент массива. Например:


```c
void func(int* arr);
```
