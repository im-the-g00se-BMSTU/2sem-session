Указатель – данные, содержащие адрес какой-либо ячейки.

Типизированный указатель - это указатель, который хранит адрес переменной определённого типа. (int\* lox)

Нетипизированный указатель - это указатель типа void\*.

Указатель на функцию хранит адрес функции и позволяет вызвать функцию через указатель.

Назначение:

1. Работа с динамической памятью

```c
int* p = malloc(sizeof(int));
*p = 10;
free(p);
```

2. Передача переменных в функцию по адресу

```c
void change(int* x) {
    *x = 100;
}

int main() {
    int a = 5;
    change(&a);
    printf("%d\n", a); // 100
    return 0;
}
```

3. Работа с массивами

```c
int arr[3] = {1, 2, 3};
int *p = arr;
printf("%d\n", *(p + 1)); // 2
```

В зависимости от системы указатель может весить 4 байта в 32-битной системе и 8 байт в 64-битной.

Операции над указателями:

1. Взятие адреса &
2. Разыменование \*, (), \[\], ->
3. Присваивание
4. Арифметические операции
5. Сравнение указателей

**Константный указатель** - это указатель, адрес в котором нельзя изменить. (значение поменять можно, int\* const p)

**Указатель на константу** - это указатель, через который нельзя изменять значение. (const int\* p)

**Константный указатель на константу** - нельзя менять ни адрес, ни значение через указатель. (const int\* const p)
