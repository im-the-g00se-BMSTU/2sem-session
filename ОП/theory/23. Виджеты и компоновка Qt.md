# 23. Qt. Основные графические виджеты. Связь кода и разметки. Компоновка виджетов.

## Qt и графические виджеты

В Qt элементы графического интерфейса называются виджетами. Виджеты - это основные строительные блоки интерфейса: кнопки, поля ввода, надписи, таблицы и другие элементы окна.

Базовым классом для всех графических виджетов является:

QWidget - То есть большинство визуальных элементов интерфейса наследуются от QWidget.

## Основные графические виджеты Qt

QLabel - Используется для отображения текста или изображения. В лекции также указано, что QLabel может использоваться как холст для вывода графики.

1. QPushButton - Обычная кнопка. При нажатии может порождать сигнал clicked().
2. QLineEdit - Поле для ввода одной строки текста.
3. QCheckBox - Флажок, который может быть включён или выключен. Является наследником QAbstractButton.
4. QComboBox - Выпадающий список для выбора одного значения из нескольких.
5. QTableWidget - Виджет для отображения таблицы.
6. QPaint - Виджет для рисования графических примитивов: точки, линии, квадрата, круга и так далее.

## Связь кода и разметки(не из лекций)

Интерфейс в Qt можно создавать двумя способами:

1. через код - вручную создавать виджеты;
2. через Qt Designer - визуально расположить элементы на форме.

Если интерфейс создаётся в Qt Designer, он сохраняется в файле: [mainwindow.ui](http://mainwindow.ui). Этот файл содержит разметку окна: какие кнопки, поля ввода, надписи и другие элементы есть на форме.

В коде связь с этой разметкой происходит через объект ui. Обычно в классе главного окна есть поле:

```c
Ui::MainWindow *ui;
```

В конструкторе вызывается:

```c
ui->setupUi(this);
```

Эта команда создаёт все виджеты, описанные в .ui-файле, и связывает их с объектом окна. После этого к элементам интерфейса можно обращаться из кода через ui-\>.

Например:

```c
ui->label->setText("Привет"); // Меняет текст надписи.
QString text = ui->lineEdit->text();// Получает текст из поля ввода.
ui->pushButton->setEnabled(false);// Отключает кнопку.
```

В лекции есть пример обращения к элементу интерфейса:

```c
param->newValue = ui->valueEdit->text().toInt();
```

Здесь ui-\>valueEdit - это поле ввода на форме, text() получает введённый текст, а toInt() преобразует его в число.

## Компоновка виджетов

Компоновка виджетов - это размещение элементов в окне. Для этого используются layout-компоновщики:

QVBoxLayout   // вертикально

QHBoxLayout   // горизонтально

QGridLayout   // сетка

QFormLayout   // форма: подпись + поле ввода
