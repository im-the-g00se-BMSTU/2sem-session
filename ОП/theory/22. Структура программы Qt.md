## Что такое Qt?

Qt - это фреймворк для разработки кроссплатформенного программного обеспечения на C/C++. Программа, написанная с использованием Qt, может быть скомпилирована под разные операционные системы без изменения исходного кода.

Qt включает большое количество готовых классов: для графического интерфейса, работы с сетью, базами данных, XML и другими задачами. Qt является объектно-ориентированным, расширяемым и поддерживает компонентный подход.

## Компонентный подход в Qt

Визуальное программирование в Qt основано на компонентном подходе. Компонент - это класс, обладающий определённым набором требований.

Компоненты бывают:

-  визуализируемые - например, кнопка, поле ввода, надпись;

-  невизуализируемые - например, таймеры, звуковые плееры.

В Qt элементы графического интерфейса называются виджетами.

QWidget - базовый класс для всех графических виджетов.

Примеры виджетов из лекции:

QLabel        // надпись

QCheckBox     // флажок

QComboBox     // выпадающий список

QPushButton   // кнопка

QLineEdit     // поле ввода

QTableWidget  // таблица

## Иерархия классов Qt

Компоненты графического интерфейса в Qt являются виджетами, а базовым классом для графических виджетов является QWidget. Классы, наследуемые от QObject или его дочерних классов, например QWidget, могут использовать сигналы и слоты.

Упрощённая иерархия классов Qt выглядит так:

1. QObject - базовый класс для большинства объектов Qt. Он обеспечивает работу с объектной моделью, событиями, сигналами и слотами.

2. QWidget - базовый класс всех визуальных элементов интерфейса.

3. QMainWindow - класс главного окна приложения.

4. QPushButton, QLabel, QLineEdit и другие - конкретные элементы интерфейса, которые наследуются от QWidget напрямую или через промежуточные классы.

## Структура программы с графическим интерфейсом(не из лекции)

Структура программы с графическим интерфейсом обычно включает:

main.cpp

mainwindow.h

mainwindow.cpp

mainwindow.ui

В файле main.cpp создаётся объект приложения и главное окно:

```c
#include <QApplication>
#include "mainwindow.h"
int main(int argc, char *argv[]) {
   QApplication app(argc, argv);//управляет работой приложения.
   MainWindow w;//создаёт главное окно
   w.show();//отображает окно.
   return app.exec();//запускает цикл обработки событий.
}
```

Класс главного окна обычно наследуется от QMainWindow:

```c
class MainWindow : public QMainWindow {
   Q_OBJECT
public:
   MainWindow(QWidget *parent = nullptr);
   ~MainWindow();
private:
   Ui::MainWindow *ui;
};
```

Файл mainwindow.ui хранит описание интерфейса, созданного в Qt Designer.\
Файлы mainwindow.h и mainwindow.cpp содержат описание и реализацию класса главного окна.
