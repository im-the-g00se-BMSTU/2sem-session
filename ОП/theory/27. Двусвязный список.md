# 27. Динамические структуры данных. Двусвязные списки. Алгоритмы их обработки.

Двусвязный список - это динамическая структура данных, состоящая из узлов, где каждый узел хранит данные, указатель на следующий узел и указатель на предыдущий узел.

То есть узел двусвязного списка содержит:

```c
typedef struct Node {
   void* data;
   struct Node* prev;
   struct Node* next;
} Node;
```

У списка обычно хранят два указателя:

```c
typedef struct List {
   Node* head; //указатель на первый элемент списка;
   Node* tail; //указатель на последний элемент списка;
   size_t dataSize;
} List;
```

Основные операции: создание узла и списка, добавление в начало и конец, удаление из начала и конца, обход вперёд и назад, очистка списка. По сравнению с односвязным списком двусвязный требует больше памяти, но позволяет удобнее работать с концом списка и перемещаться в обе стороны.
